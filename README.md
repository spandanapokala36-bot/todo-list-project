# To-Do List App

A simple interactive to-do list application built using HTML, CSS, and JavaScript.

## Live Demo
https://spandanapokala36-bot.github.io/todo-list-project/

## Features
- Add tasks
- Delete tasks
- Add tasks using Enter key
- Dark / Light mode toggle
- Input validation

- JavaScript
